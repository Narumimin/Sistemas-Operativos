COMO INICIAR EL PROGRAMA

ES IMPORTANTE ACLARAR QUE ESTO SOLO SIRVE EN UN ENTORNO LINUX DEBIDO A UNAS CUANTAS LIBRERIAS QUE SE USAN EN EL CODIGO
TAMBIEN SE HIZO USANDO VISUAL STUDIO CODE ASI QUE ES IMPORTANTE TENERLO DESCARGADO, JUNTO CON LA EXTENSION DE C++ Y LA EXTENSION CODE RUNNER

PASOS PARA USAR EL CODIGO:
1. DESCARGAR TODOS LOS ARCHIVOS .cpp Y .h EN UN MISMO DIRECTORIO, PREFERIBLEMENTE UNO NUEVO
2. CORRER EL CODIGO A PARTIR DEL ARCHIVO "Main.cpp"

Y LISTO ESO DEBERIA SER TODO

SI HAY PROBLEMAS AL EJECUTAR EL CODIGO CONTACTARSE CON EL EQUIPO E INTENTAREMOS AYUDAR LO MAS RAPIDO QUE SE PUEDA
